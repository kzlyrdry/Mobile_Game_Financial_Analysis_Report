Mobile Game Financial Analysis Report
Project Overview

This project focuses on analyzing mobile game user behavior and monetization performance by transforming raw data into a structured, analytics-ready reporting system. The goal was to design a scalable BI solution that supports executive-level decision-making.

Tools & Technologies

Python (Pandas, NumPy)

Power BI

DAX

Data Warehouse Modeling

Data Preparation & Feature Engineering

Cleaned and processed raw monetization data, treating missing purchase values as non-paying users

Created behavioral user segments (Whale, Dolphin, Minnow) based on total spending

Calculated DaysSinceLastPurchase to support churn and retention analysis

Generated analytical features such as AgeGroup and ActivityLevel

Prepared a clean dataset optimized for Power BI modeling

Data Modeling & KPIs

Designed a relational data model aligned with data warehouse standards

Implemented all KPIs using DAX, ensuring consistency and flexibility

Key metrics include ARPU, ARPPU, Conversion Rate, Total Revenue, and user segmentation metrics

Reporting & Insights

Built executive-level dashboards covering financial KPIs and revenue trends

Analyzed monetization performance across geographies and user segments

Provided retention and user value insights using segmentation and activity analysis
